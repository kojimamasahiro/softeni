# 2026 ハイスクールジャパンカップ — 大会結果ツール入力JSON

第55回ゴーセン杯争奪ハイスクールジャパンカップソフトテニス2026（北海道、2026/6/25〜6/28）。
発表された組み合わせ表（公式トーナメント表 PDF: `singles.pdf` / `doubles.pdf`）から、
大会結果ツールへ渡す入力JSON（`initialPlayers` 形式）を種目別に作成。

## 仕様

- フラット配列。`tools/tournament3` 等の `initialPlayers` 形式。
- **配列順（`id`）= ドロー表の番号（上から下の並び）**。隣り合う 2 件が 1 回戦の対戦。
- `team` = 所属校、`prefecture` = 都道府県。
- `tempId`
  - シングルス: `姓_名_所属_都道府県`
  - ダブルス: `姓_名_所属_都道府県`（各選手ごと。同姓ペアは末尾 `_2` で区別）
- `name` は表示用ラベル（`姓 名（都道府県・所属）` / `姓・姓（都道府県・所属）`）。

## bye（1回戦免除）の扱い

- シングルスは 64 ちょうど（2 の冪）なので bye 不要。
- ダブルスは 72 のため bye が必要。**入力JSONには bye を入れていない**（番号順そのまま）。
  ツール上で「パッキン」機能を使って 1 回戦免除を設定すること（組み合わせ表の線に合わせる）。

## フルネーム補完について

- ダブルスの `firstName` は、出場校一覧
  （https://www.gosen.jp/racketsports/hjs/participant/2026/ の 男子ダブルス / 女子ダブルス タブ）
  の名簿と **姓で照合して補完済み**（男子72ペア・女子72ペア＝各144人）。異体字は照合時のみ正規化し、
  出力の姓はトーナメント表どおりに保持。
- **要確認 1 件**: 女子ダブルス No.52 東北（宮城県）`柏・村上` の **村上** は `firstName` 空。
  公式名簿が「柏 春花／柏 春花」と重複表記のため、村上選手の名が判別不能。手動確認が必要。

## 既知の限界

- 氏名・所属は PDF テキストレイヤーから座標ベースで抽出（シングルスの姓名分割は字間ギャップで判定）。
  異体字（髙・﨑・𠮷・栁・瀨 等）はそのまま保持。最終確認を推奨。
