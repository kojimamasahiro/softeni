アカウント削除

このページでは、アプリを再インストールせずにアカウント削除を実行できます。

削除手続き

まずログインして本人確認を行ってください。ログイン後、このページ上で削除確認と削除実行まで完了できます。

プライバシー情報は プライバシーポリシー を確認してください。

削除対象

  • users に保存されたプロフィール設定
  • oauth_tokens の連携トークン
  • earnings_daily と analytics_daily の日次データ
  • daily_summary と異常検知ログ
  • fcm_tokens と監視対象 app / target データ

保持する可能性があるデータ

不正防止、課金照合、法令対応のため、subscription / entitlement / reward 関連の一部記録を必要な期間だけ保持する場合があります。
