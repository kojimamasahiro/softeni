# アイデア: 高校ソフトテニス 強豪校ランキングページ

## 状況

Idea Backlog。検討中（2026-07-17 起票）。「ソフトテニス 高校」ヘッドクエリ対策の SERP 調査から派生。

## 目的

「ソフトテニス 高校 強豪」「高校 ソフトテニス ランキング」系クエリの受け皿となる専用ページを作り、
高校カテゴリの検索露出を増やす。ヘッドクエリ「ソフトテニス 高校」の主要サブ意図のうち、
当サイトが唯一データ優位で戦える領域（高体連公式は自県の結果のみ、競合ランキングサイトはデータが薄い）。

## ユーザーが興味を持った点

「中期の案はIdea Backlogに記録してほしい。また、この案はどういったものか？効果はありそうなのか」（2026-07-17）
— タイトル修正の即実施と合わせて記録を依頼。効果の見立てに関心。

## 想定運用 / なぜこのスコープか

- 主要4大会（インターハイ・国体・ハイジャパ・選抜）の成績 DB から学校ごとにポイント化
  （例: 優勝/準優勝/ベスト4/ベスト8 に傾斜配点、直近3年に重み）し、全国＋都道府県別のランキングを表示。
- URL は 1 ページ（または男女 2 ページ）に集約し、年度・県別はクライアント切替。
  `/rankings/`（選手ランキング）が確立した「薄いページを量産しない 1 URL 方式＋静的 HTML の上位まとめ」
  パターンをそのまま踏襲する（seo.md #9 と同型）。
- 各校名から既存の学校ページ（`/highschool/{gender}/{prefectureId}/{teamId}`）へ内部リンク → 学校ページ群への評価波及。

## わかっていること

- SERP 調査（2026-07-17）: 「ソフトテニス 高校 強豪 ランキング」の上位は iezo.net・がくらん・
  受験系ブログ（埼玉県立高のトリセツ、東京スクールデータ等）。いずれも成績データの網羅性・更新頻度が低く、
  県単位の記事や自己申告ベースの集計。当サイトの大会成績 DB（主要4大会×年度別×全国）が質で上回れる。
- 受験系ブログが単発記事で上位に入れている＝ドメイン権威が低くてもコンテンツ一致で戦えるクエリ帯。
- 集計に必要なデータ・判定ロジックは既存実装にほぼ揃っている
  （`lib/highschool.ts` の大会優先度・ベスト8判定、`data/highschool/prefectures/*/summary.json`、
  歴代ページの `buildChampionSummary` 等）。
- Assumption: 検索ボリュームの実数は未計測（Trends の相対値と SERP の競合状況からの推定）。

### データとして追加する価値があるもの（優先度順・評価）

1. **選抜の過去年度**: ◎ ランキング精度に直結。データ形式は既存 2025 と同じで取り込みコスト低
2. **国体/国スポ 少年の部**: ◎ 4大会構想の欠け。学校サマリー・ランキング両方に効く
3. **学校属性（公立/私立・所在地）**: ○ ランキングのフィルタ・受験系意図（競合の主戦場）への差別化。軽量
4. **地方ブロック大会（東海選抜等の各地区版）**: △ 準強豪校の解像度は上がるが量が多い。県別展開が当たってから
5. **都道府県予選**: × 当面非推奨。量が膨大で高体連公式と正面競合、ランキングへの寄与も小さい
