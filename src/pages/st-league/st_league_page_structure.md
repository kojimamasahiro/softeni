# STリーグ 対戦結果ページ仕様書（構成案①）

## 1. ページの目的

- STリーグ（8チームまたは7チーム総当たり戦）の**その年の全対戦結果を1ページで把握できる**こと
- スマートフォン閲覧を最優先とし、
  - 一覧性
  - 操作の少なさ
  - 情報過多にならない構造
    を重視する
- 分析・個人成績などの詳細データは**別ページに集約**し、本ページでは扱わない

---

## 2. 想定ユーザー

- STリーグ出場チームの関係者・選手
- 特定チームを応援しているファン
- 「今年のSTリーグはどうだったか」を俯瞰したいソフトテニス経験者

---

## 3. 大会前提条件（仕様に影響する前提）

- 出場：8チームまたは7チーム
- 形式：総当たり戦（各チーム7試合または6試合）
- 各対戦：3試合制
  - ダブルス① → シングルス → ダブルス②
- 勝敗決定：3試合中2勝先取

---

## 4. ページ全体構成（1ページ完結）

```
[ ページタイトル ]
[ 性別切り替え ]

[ 順位サマリー（固定） ]

[ 対戦結果一覧（展開式） ]

[ データ・分析ページへの導線 ]
```

---

## 5. 各セクション詳細

### 5.1 ページヘッダー

- タイトル例
  - `STリーグ 2025 試合結果・日程
- サブ情報
  - 開催日程
  - 男子 / 女子 タブで切り替え

※ スマホではシンプルに1行表示

---

### 5.2 順位サマリー（常時文脈用）

#### 表示内容

| 項目     | 内容     |
| -------- | -------- |
| 順位     | 1〜8位   |
| チーム名 | 公式名称 |
| 勝敗     | ○勝×敗   |
| 本数     | 例：15-6 |

#### 表示例（スマホ）

```
1位 Aチーム   6勝1敗  15-6
2位 Bチーム   5勝2敗  13-8
3位 Cチーム   4勝3敗
...
```

- スクロール後も「今どの年・大会を見ているか」が分かる
- JSで簡易的に固定（sticky）しても良い

---

### 5.3 対戦結果一覧（メインコンテンツ）

#### 基本構造

- 1対戦 = 1行（初期表示）
- タップで3試合の内訳を展開

#### 初期表示（折りたたみ）

```
Aチーム vs Bチーム   ○ A（2-1）
```

表示要素：

- チーム名 ×2
- 勝敗（○×）
- 対戦スコア（2-1 / 3-0 など）

---

#### 展開後表示（詳細）

```
Aチーム vs Bチーム   ○ A（2-1）

D1  ○ 4-2
S   × 2-4
D2  ○ 4-1
```

- 試合順は必ず
  - ダブルス① → シングルス → ダブルス②
- 個人名・戦術要素は表示しない

---

### 5.4 フィルター・補助UI（任意）

#### 想定フィルター

- チーム指定（特定チームのみ表示）
- 勝敗（勝利 / 敗戦）

※ すべて**ページ内JS制御**で完結

---

### 5.5 データ・分析ページへの導線

#### 役割

- 本ページでは「結果」まで
- 深掘りしたいユーザーを別ページへ送る

#### 導線文言例

```
▶ チーム別・選手別の詳細データを見る
```

リンク先例：

- `/st-league/2026/data`
- `/st-league/2026/analysis`
